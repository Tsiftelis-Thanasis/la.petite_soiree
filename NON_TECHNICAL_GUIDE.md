## Initial Setup (One-Time, May Need Help)

### Prerequisites:

You need:
1. An email address
2. The website to be deployed (see `HOW_TO_DEPLOY.md`)
3. An invitation to edit the site (sent by site admin)

### Getting Access:

**Step 1: Receive Invitation Email**
- You'll get an email titled "You've been invited to join..."
- Click **"Accept the invitation"**

**Step 2: Set Your Password**
- Create a secure password
- Click **"Sign up"**

**Step 3: Save Your Login**
- Write down: `https://your-site-name.netlify.app/admin/`
- Bookmark this page in your browser

That's it! You're ready to edit.

## Writing Blog Posts

**What it controls:** Blog posts and portfolio entries showcasing past events

### To Create a New Post:

1. **Click:** "Blog & Portfolio"

2. **Click:** "New Blog & Portfolio" (top right)

3. **Fill in:**

   - **Title:** e.g., "Elegant Garden SoirÃ©e"

   - **Publish Date:** Click calendar icon, select date and time

   - **Featured Image:** Upload a hero image for the post

   - **Description:** A short summary (shown in previews)

   - **Body:** The main content (use the formatting toolbar)
     - Bold, italic, links
     - Headers, lists
     - Insert images

   - **Tags:** Add keywords (press Enter after each)
     - e.g., "wedding", "garden", "outdoor"

   - **Published:**
     - âœ… Checked = visible on website
     - âŒ Unchecked = draft (not visible)

4. **Click:** "Publish" â†’ "Publish now"

### Body Editor Tips:

The body editor supports **rich text**:

- **Bold:** Select text, click B
- **Italic:** Select text, click I
- **Link:** Select text, click link icon, paste URL
- **Heading:** Click H2 or H3
- **List:** Click bullet or number icon
- **Image:** Click image icon, upload
- **Quote:** Click quote icon

### To Save as Draft:

1. Uncheck "Published" toggle

2. Click "Save"

3. Post won't appear on website yet

### To Publish a Draft:

1. Open the draft post

2. Check "Published" toggle

3. Click "Save"

## Uploading Images

### Image Requirements:

- **Format:** JPG or PNG
- **Size:** Under 10MB
- **Dimensions:** Recommended 1920x1080 or similar

### How to Upload:

**Method 1: During Editing**

1. Click on an image field

2. Click "Choose an image"

3. **Drag and drop** your image file
   - OR click "Upload" and select file

4. Image uploads and appears

**Method 2: Media Library**

1. Click "Media" (if available in menu)

2. Upload images here

3. Use them later in any content

### Image Best Practices:

- **File names:** Use descriptive names (e.g., `garden-wedding-setup.jpg`)
- **Size:** Compress large images before uploading
  - Use [TinyPNG](https://tinypng.com/) to reduce file size
- **Alt text:** Always add alt text for accessibility

## Seeing Your Changes

### After Clicking Save:

1. **Wait 1-2 minutes** for changes to go live

2. **Open your website** in a new tab: `https://your-site-name.netlify.app/`

3. **Refresh the page** (press F5 or Ctrl+R)

4. **See your changes!**

### If Changes Don't Appear:

- Wait a bit longer (up to 5 minutes)
- Clear browser cache:
  - Press **Ctrl + Shift + Delete** (Windows/Linux)
  - Press **Cmd + Shift + Delete** (Mac)
  - Select "Cached images and files"
  - Click "Clear data"
- Try opening in incognito/private window

## Tips for Great Content

### Writing Style:

- **Be conversational:** Write like you're talking to a friend
- **Be specific:** Instead of "we plan events," say "we plan intimate garden weddings and elegant dinner parties"
- **Tell stories:** Share details about real events you've created

### Image Tips:

- **High quality:** Use professional photos
- **Consistent style:** Photos should have similar lighting/vibe
- **Show details:** Close-ups of table settings, dÃ©cor, flowers
- **Before/after:** Show transformations

### SEO (Search Engine Optimization):

- **Use keywords naturally:** "event planning," "wedding styling," "Paris-inspired"
- **Write descriptive titles:** Not just "My Post" but "Elegant Garden Wedding in Paris Style"
- **Add alt text to images:** Helps search engines and accessibility

## Troubleshooting

### "Can't access /admin/"

**Solutions:**
- Check you're using the correct URL
- Make sure you're logged in
- Try clearing browser cache
- Try a different browser

### "Authentication error"

**Solutions:**
- Check your email/password are correct
- Click "Forgot password" to reset
- Contact the site admin to resend invitation

### "Changes not saving"

**Solutions:**
- Check your internet connection
- Make sure you clicked "Save" or "Publish"
- Try refreshing the page and saving again
- Check for error messages (red text)

### "Image won't upload"

**Solutions:**
- Check file size (under 10MB)
- Check format (JPG or PNG only)
- Try compressing the image first
- Check your internet connection

### "Can't find my post"

**Solutions:**
- Check if "Published" is checked (toggle on)
- Wait 2-3 minutes for deployment
- Clear browser cache and refresh
- Check if you're looking at the right page
